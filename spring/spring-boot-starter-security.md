# spring-boot-starter-security


## Spring Security 
  - Spring 기반의 애플리케이션의 보안을 담당하는 스프링 하위 프레임워크
  - Spring Security는 Servlet `Filter`를 기반
  - 이 `Filter`들을 통해 Authentication을 처리
